## 🏠 داشبورد

صفحه اول که بعد از ورود می‌بینی:

| کارت | توضیح |
|------|-------|
| 👥 مشتریان | تعداد کل مشتریانت |
| 📋 سفارش امروز | چند نفر امروز وقت داری |
| 💰 درآمد امروز | مبلغ پرداخت‌شده امروز |
| 📈 درآمد ماه | جمع درآمد ماه جاری |

**یادآوری سرویس:** اگه برای مشتری "سرویس بعدی" تعریف کرده باشی،
اینجا نشون میده که چه کسی باید در ۳۰ روز آینده سرویس بشه.

## 👥 مدیریت مشتریان

### ثبت مشتری جدید:
1. روی دکمه **+** گوشه بالا کلیک کن
2. نام، موبایل، آدرس و نوع دستگاه رو وارد کن
3. ذخیره

### جستجو:
بار جستجو بالای صفحه هست — با نام، موبایل یا آدرس جستجو کن.

### محدودیت رایگان:
- نسخه رایگان: **۵۰ مشتری**
- نوار پیشرفت بالای لیست نشون میده چقدر استفاده کردی
- برای بیشتر از ۵۰ نفر → **اشتراک طلایی**

---

## 📋 مدیریت سفارش‌ها

### ثبت سفارش:
1. دکمه **ثبت سفارش جدید** (سبز)
2. نام مشتری رو تایپ کن
3. نوع خدمات رو انتخاب کن (از لیست آماده)
4. تاریخ مراجعه رو بنویس
5. وضعیت: در انتظار / در حال انجام / تکمیل / لغو

### تغییر وضعیت سریع:
زیر هر سفارش، دکمه‌های کوچک هستن که بدون کلیک اضافه وضعیت رو عوض میکنن.

### یادآوری سرویس دوره‌ای:
توی هر سفارش، **تاریخ سرویس بعدی** رو بنویس.
داشبورد بهت یادآوری میکنه که سرویس کدوم مشتری نزدیکه.

---

## 🧾 فاکتور

### ایجاد فاکتور:
1. دکمه **ایجاد فاکتور جدید** (بنفش)
2. مشتری رو انتخاب کن
3. اقلام رو اضافه کن:
   - **شرح:** مثلاً "فیلتر پکیج" یا "گاز R22"
   - **تعداد:** عدد
   - **قیمت واحد:** به تومان
4. اجرت، تخفیف، مالیات رو وارد کن
5. جمع کل خودکار محاسبه میشه
6. ذخیره

### چاپ PDF:
روی هر فاکتور کلیک کن → دکمه **🖨️ چاپ PDF**

### ثبت پرداخت:
روی فاکتور کلیک کن → دکمه **ثبت پرداخت**
- پرداخت کامل → وضعیت: ✅ پرداخت شده
- پرداخت ناقص → وضعیت: 🔸 پرداخت ناقص

## ⚙️ تنظیمات

- **نام تعمیرکار:** اسمی که روی فاکتور چاپ میشه
- **آدرس:** آدرسی که روی فاکتور ظاهر میشه
- **تخصص:** نوع کارت (پکیج، کولر، برق...)
- **حالت تاریک:** شب راحت‌تر کار کن
- **رنگ برنامه:** ۶ رنگ مختلف انتخاب کن
- **یادآوری SMS:** فعال/غیرفعال

## 🗄️ مدیریت دیتابیس (phpMyAdmin)

برای مدیران سیستم:

1. وارد **cPanel → phpMyAdmin** شو
2. دیتابیس `servisyar_db` رو انتخاب کن

### بکاپ:
`Export → Quick → SQL → Go`
→ فایل .sql رو دانلود و جای امنی نگه دار

### بازگردانی:
`Import → فایل .sql رو انتخاب کن → Go`

### مشاهده داده:
هر جدول رو کلیک کن تا ببینی مشتریان، سفارش‌ها، فاکتورها چی هستن.

---

## 🔄 نسخه‌برداری خودکار

برای بکاپ خودکار دیتابیس، از **cPanel → Backup Wizard** استفاده کن.
یا یه cron job بذار که هر شب یه backup بگیره:

```
0 2 * * * mysqldump -u USER -pPASS DB_NAME > /home/user/backups/servisyar_$(date +%Y%m%d).sql
```

## 📞 سوالات متداول

**سوال: آیا برنامه به اینترنت نیاز داره؟**
جواب: برای استفاده روزمره **نه** — همه داده‌ها روی سرور خودته. فقط برای باز کردن سایت نیاز به اینترنت هست.

**سوال: اگه هاست قطع بشه چی؟**
جواب: دیتا حفظ میشه، فقط به برنامه دسترسی نداری تا هاست برگرده.

**سوال: چند نفر میتونن همزمان استفاده کنن؟**
جواب: هر تعمیرکار اکانت جداگانه داره. روی یه هاست میشه چندین تعمیرکار داشت.

**سوال: آیا اطلاعات ما امنه؟**
جواب: بله — داده‌ها روی سرور خودت ذخیره میشه، نه سرور ما یا هیچ شرکت خارجی.
